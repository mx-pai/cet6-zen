# CET-6 Zen 📚

> 一个优雅的 CET-6 英语六级离线做题应用，支持 PDF 原生渲染、笔记标注、答题记录管理。

## ✨ 特性

### 📖 试卷管理

- **在线试卷库** - 内置 36 套历年真题（2020-2025），一键导入
- **书架式布局** - 试卷以彩色书脊的形式陈列在木质书架上，直观美观
- **年份分组** - 自动按年份分类试卷，支持折叠/展开
- **批量导入** - 一次性导入多份 PDF 真题，自动识别年份
- **完成进度** - 实时显示每份试卷的完成百分比

### ✍️ 做题体验

- **原生 PDF 渲染** - 使用 PDF.js 高清渲染，支持 HiDPI/Retina 屏幕
- **多彩标注工具** - 5 种笔色（红/蓝/绿/紫/橙）+ 高亮 + 橡皮擦
- **全屏答题区** - 作文、翻译、草稿区域自动占满可用空间
- **标准答题卡** - 听力 25 题、阅读 30 题（选词填空/段落匹配/仔细阅读）
- **智能计时** - 自动记录做题时间，支持暂停/继续

### 💾 数据管理

- **离线优先** - 所有数据存储在本地（LocalStorage + IndexedDB）
- **自动保存** - 答案和标注实时保存，永不丢失
- **状态持久化** - 书架折叠状态、答题进度全部保留

### 🎨 界面设计

- **现代 UI** - Tailwind CSS + 自定义动画
- **响应式** - 完美适配桌面端和移动端
- **流畅交互** - Sticky 导航、平滑滚动、视觉反馈

## 🌐 部署

### 部署到 Vercel（推荐）

1. Fork 或 clone 本仓库到你的 GitHub 账号
2. 访问 [Vercel](https://vercel.com)，使用 GitHub 登录
3. 点击 "New Project"，选择本仓库
4. 配置如下：
   - Framework Preset: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. 点击 "Deploy"，等待部署完成

部署成功后，你将获得一个在线地址，例如：`https://cet6-zen.vercel.app`

### 在线试卷库

本项目已内置 **36 套历年真题**（2020-2025），无需本地上传：

1. 进入应用，点击右上角 **"在线试卷库"** 按钮
2. 浏览按年份分组的真题列表
3. 点击 **"导入"** 按钮，即可将试卷添加到你的题库
4. 开始做题！

所有试卷均存储在 `public/` 目录，索引文件为 `public/papers.json`。

详细部署指南请查看：[DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md)

## 🎯 使用指南

### 1. 导入试卷

点击右上角"批量导入真题"按钮，选择多个 PDF 文件。系统会：

- 自动识别文件名中的年份（如 2023、2024）
- 生成书架上的彩色书脊卡片
- 存储 PDF 到 IndexedDB

### 2. 开始做题

点击书架上的任意试卷进入做题模式：

- 左侧：PDF 查看器 + 标注工具
- 右侧：答题区域（作文/听力/阅读/翻译/草稿）

### 3. PDF 标注

工具栏功能：

- **钢笔**: 在 PDF 上画线标记
- **高亮**: 荧光笔效果
- **橡皮擦**: 点击标记删除
- **颜色**: 5 种笔色可选
- **缩放**: 调整 PDF 大小
- **翻页**: 左右箭头或键盘 ← →

### 4. 答题卡

- **作文/翻译**: 大面积文本输入框，自动词数统计
- **听力**: 25 道选择题（A/B/C/D）
- **阅读**:
  - 选词填空 (26-35): 15 选 10
  - 段落匹配 (36-45): 文本输入框，支持任意选项字母（如 A-J、A-M、A-O）
  - 仔细阅读 (46-55): 10 道选择题
- **草稿**: 随手记笔记、单词、长难句

### 5. 返回题库

点击右上角"返回题库"，系统会：

- 提示确认（避免误操作）
- 自动保存所有答案
- 返回书架界面
